Zoo Booking System

A secure PHP + MySQL system with daily limits, loyalty points, and compliance features.

Features

  • Daily ticket limit enforcement
  • Multiple ticket types (Family, Adult, Student, Child)
  • Loyalty points awarded per booking
  • Cookie consent compliance

Design Process

To ensure the Zoo Booking System met diverse user needs, I created personas and empathy maps for four key groups.

New User

First-time visitor who needs clear guidance and simple booking steps.

Existing User

Returning visitor who values loyalty points and quick repeat bookings.

Teacher

Organises educational visits, needs group booking options and compliance features.

Admin User

Manages bookings, monitors availability, and ensures system security and compliance.
